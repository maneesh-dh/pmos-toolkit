Risks

  • PDF tabulation reliability. Mitigation: ship Typeform + Google Forms fixtures; document explicit best-effort + fallback.
  • SKILL.md body bloat. 8 phases + canonical non-interactive block + platform-adaptation + anti-patterns risks >500 lines. Mitigation: push detail to reference/ aggressively; review at T24 and split if needed.
  • Subagent contract drift. Phase 5's subagent contract is text-prompt-defined; small wording changes can shift outputs. Mitigation: lock the contract in reference/text-analysis.md as a verbatim template the SKILL.md body cites.
  • Helper version skew with /survey-design. Both skills share a survey-mental-model but no code. Mitigation: spec confirms /survey-design's survey.json sections[].questions[] shape is what we consume.

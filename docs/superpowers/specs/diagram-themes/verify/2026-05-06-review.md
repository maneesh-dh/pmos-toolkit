## Bugs fixed during /verify

### V3-1 — SVG path parser IndexError on odd-length coords (pre-existing, conf 92)

`tests/run.py` `evaluate()` parses path `d=` strings into waypoints. M and L commands iterated `nums[i+1]` without bounds-checking; an odd-length coordinate list (e.g., `M 10 10 20`) raised `IndexError` and crashed the entire selftest runner.

**Pre-existing** — the bug was in run.py before any Phase 1-3 work. Defect SVGs in the corpus happen to use only well-formed paths.

**Fix:** loop range capped at `len(nums) - 1` and `len(nums) < 2` early-return for M.

**Regression:** `test_evaluate_handles_odd_length_path_M_command` in `test_theme_loader.py` constructs an SVG with `M 10 10 20` and asserts `evaluate()` returns a dict instead of crashing.

### V3-2 — Caption grid 5-column layout 2px slack at right edge (T17 bug, conf 88)

`wrapper/caption_grid.py:caption_layout(5, ...)` computed `narrow_w = (usable - 4*gutter) // 6` and `wide_w = 2 * narrow_w`. Integer division of `1088 / 6 = 181.33` rounded down to 181, leaving 2px unused at the canvas right edge — the rightmost caption ended at x=1214 instead of x=1216.

**Fix:** distribute the integer-division slack into the wide column: `wide_w = 2 * narrow_w + (usable - 6*narrow_w - 4*gutter)`.

**Regression:** `test_five_captions_consume_full_usable_width` asserts the rightmost column's right edge equals `total_width - margin` exactly.

The original `widths[2] == 2 * widths[0]` test was loosened to `2*narrow ≤ wide ≤ 2*narrow + gutter` to accommodate the slack-redistribution contract.

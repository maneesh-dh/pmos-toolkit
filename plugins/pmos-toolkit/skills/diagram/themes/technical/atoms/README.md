# Style Atoms — Visual Vocabulary Reference

These are NOT templates. The agent never copies structure from them. They exist solely to calibrate visual tone — stroke weights, color, font sizing, spacing density, corner radii.

When generating a diagram the agent should glance at these to confirm "is what I'm about to draw recognizably from the same author?" — not "what shape should I draw?".

| File | Atom |
|---|---|
| `01-node-primary.svg` | Single primary node — 8px radius, 2px ink stroke, 16/600 ink label |
| `02-node-default.svg` | Default node — 4px radius, 1.5px ink-muted stroke, 14/400 ink label |
| `03-connector-default.svg` | Orthogonal connector with arrowhead — 1.5px ink-muted, marker `arrow` |
| `04-connector-emphasis.svg` | Emphasis connector — 2px accent stroke |
| `05-edge-label.svg` | Edge label pill — 12/400 ink-muted on surface-muted, 4px radius |
| `06-legend.svg` | 3-row legend block — 16×16 swatches, 12/400 labels |
| `07-swatches.svg` | Color swatch sheet — all 6 tokens with names and hex codes |
| `08-type-specimen.svg` | Type specimen — 12/14/16/20 at 400 and 600 weights |
